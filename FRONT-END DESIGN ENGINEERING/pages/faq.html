Perguntas Frequentes

Tire suas dúvidas sobre a plataforma OrbitSafe.

O que acontece se as antenas forem danificadas?

Caso uma antena fique indisponível, o OrbitSafe utiliza a comunicação via satélite para manter o envio de informações. Quando ocorre uma falha na comunicação, o sistema prioriza rotas alternativas disponíveis para manter a transmissão dos dados, reduzindo o impacto da interrupção. Assim, o monitoramento continua funcionando mesmo em situações de emergência.

Como o chatbot funciona sem internet?

Mesmo sem acesso a uma rede de internet, os dados são transmitidos através dos satélites de órbita baixa, que permitem que essa comunicação possa ser mantida por redes via satélite ou outras infraestruturas de emergência.

Quanto tempo leva para detectar uma queda de rede?

A detecção é praticamente imediata. O sistema monitora constantemente a comunicação e identifica interrupções em poucos segundos, permitindo uma resposta rápida de aproximadamente 30 segundos.

Quem pode acessar o Dashboard?

O Dashboard é destinado a órgãos responsáveis pelo monitoramento, como equipes de emergência, Defesa Civil e administradores autorizados, garantindo acesso seguro às informações.

Quais satélites são utilizados?

O OrbitSafe é um projeto idealizado para ser compatível com redes de comunicação via satélite utilizadas em monitoramento e resposta a desastres, como os Satélites de órbita terrestre baixa (LEO).

É possível integrar com a Defesa Civil?

Sim. O sistema foi projetado para facilitar a integração com órgãos públicos, permitindo o compartilhamento de alertas e informações que auxiliem nas operações de resposta e resgate.
